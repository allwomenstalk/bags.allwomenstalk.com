# 7 Colour Block Bags ...

Colour block bags are a simple way to add a contemporary twist to an outfit. They’re also an ideal way to incorporate the colour blocking trend into your wardrobe. For maximum impact, choose bags with clashing colours. For something more subtle, a bag with a neutral block colour is an ideal way to tone it down. Check out the following colour block bags.

## 5. ASOS Patent Twist Lock Saddle Bag
